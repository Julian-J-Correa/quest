# The PHP Questionaire
This questionaire was created for the purpose of understanding how to use .php to create a website. This website was hosted using XAMPP with Apache as the server and MySQL as the database system.

## Pages
- Main Menu
- Login
- Questionaire (consisting of 3 pages including confirmation of completion
- Admin Dashboard
- Users table
- About page
## Functions
- Fill in the questionaire and have it saved in the database
- View and delet questionaire results
- Add, edit, delete and view verified users of the website
